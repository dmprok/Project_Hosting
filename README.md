**Рекомендации по работе с репозиторием:**
1. Скачайте все файлы из репозитория и расположите в одной папке с данными. Данные имеют следующий формат: папки *train_images*, *test_images*; файлы *test_images.csv*, *test_queries.csv*, *train_dataset.csv*, *CrowdAnnotations.tsv*, *ExpertAnnotations.tsv*).
2. Основной проект находится в файле *project_hosing.ipynb*.
3. В файлах csv сохранены результаты обучения моделей, в файле pth сохранены веса лучшей нейросети. Эти файлы не являются обязательными, но их наличие позволит пропустить длительное обучение моделей.
4. В файле *requirements.txt* находятся подробные требования к окружению. **ВАЖНО!** модели в проекте обучаются на GPU с помощью библиотек RAPIDS(cuml, cupy), выполнение проекта возможно при наличии настроенного CUDA в системе.
